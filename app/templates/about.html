About the Pizza Management System

Project Overview

This Pizza Management System is a comprehensive web application built using Flask and MySQL, demonstrating professional database design and full-stack web development practices. The application provides complete CRUD (Create, Read, Update, Delete) operations across multiple related database tables.

Features

  • Customer Management: Track customer information including names, phone numbers, and email addresses
  • Pizza Inventory: Manage pizza varieties with different sizes, pricing, and cost tracking for profit analysis
  • Order Processing: Create and manage orders with multiple line items, quantities, and automatic total calculations
  • Order Details: Track individual items within each order with pizza selection and quantity management

Technical Stack

  • Backend: Python Flask framework
  • Database: MySQL with foreign key relationships
  • Frontend: Bootstrap 5 with responsive design
  • UI Components: Modal dialogs and dropdown menus for enhanced UX

Database Schema

The system uses a normalized relational database with five interconnected tables:

  • Customer: Stores customer information
  • Pizza: Contains pizza varieties with pricing and cost data
  • Order: Records order header information linked to customers
  • Order_Detail: Maintains line items for each order with quantities
  • Employee: Manages user authentication and access
